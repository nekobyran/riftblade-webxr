## 1. WebXR 节奏光剑需求

### 用户

> 网页端复刻 VR 版的节奏光剑，越还原越好。自己创作三首不同风格的曲目，每首歌的关卡环境需要符合歌曲意境，背景最好是动态画面，受伤的光剑也要符合不同主题。使用 WebXR 技术实现，让我可以用 VR 游玩，并部署到我的 GitHub 静态托管。

### Codex

已创建原创 WebXR 节奏双剑游戏 **RIFT//BLADE 光痕裂界**，并完成 GitHub Pages 部署。

- 在线体验：[RIFT//BLADE 光痕裂界](https://nekobyran.github.io/riftblade-webxr/)
- 源码仓库：[nekobyran/riftblade-webxr](https://github.com/nekobyran/riftblade-webxr)
- 部署流水线：[GitHub Actions](https://github.com/nekobyran/riftblade-webxr/actions/runs/29641085393)

主要能力：

- WebXR `immersive-vr` 会话和双手控制器光剑。
- 手别、挥砍方向、接触距离和节拍时机判定。
- 分数、连击、倍率、生命、暂停、失败结算和重新开始。
- Quest 等头显的控制器触觉反馈，并提供桌面键鼠试玩模式。
- 完整选曲、教学、倒计时、HUD、受伤提示和结算界面。
- 响应式布局、键盘可访问性和 `prefers-reduced-motion` 支持。

## 2. 三首原创曲目与主题世界

### Neon Tide Run / 霓虹潮汐

- 132 BPM，约 78 秒。
- 风格：Liquid Synthwave。
- 场景：雨幕数据堤、霓虹城市、跃迁环和动态均衡器建筑。
- 受伤剑效果：电涌、色差闪烁和浪花粒子。

### Ember Circuit Choir / 余烬回路圣咏

- 104 BPM，约 84 秒。
- 风格：工业仪式慢拍。
- 场景：地下熔炉圣堂、活塞拱门、熔岩缝隙和上升余烬。
- 受伤剑效果：熔岩裂纹、炽热闪光和余烬爆裂。

### Glass Orbit Monsoon / 玻璃轨道季风

- 148 BPM，约 92 秒。
- 风格：零重力 Tabla Drum & Bass。
- 场景：轨道雨园、六边形光环、旋转玻璃碎片和棱镜雨。
- 受伤剑效果：晶体破碎、彩色分光和棱镜粒子。

三首音乐均由浏览器实时程序化合成，不依赖第三方歌曲或受版权保护的音乐素材。
